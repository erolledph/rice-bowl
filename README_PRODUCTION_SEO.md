### For Daily Recipe Posts
1. Create recipe in `app/recipes/recipe-name.md`
2. Use proper frontmatter (title, description, image, etc.)
3. File name becomes the URL slug (kebab-case)

### For Best SEO Results
- Write descriptive titles (include main ingredient)
- Use high-quality images (1200x630px minimum)
- Write natural, complete descriptions
- Use all metadata fields (prep time, difficulty, etc.)
- Add relevant ingredient and cuisine tags
- Write clear, numbered instructions
